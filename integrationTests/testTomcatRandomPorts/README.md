# testTomcatContextConfig

Example of configuring webapp context with randomly-selected HTTP and HTTPS ports.

## How to run

```bash
cd examples/testTomcatRandomPorts
gradle appRun
```

## How to test

```bash
cd examples/testTomcatRandomPorts
gradle integrationTest
```

## How to build a product


```bash
cd examples/testTomcatRandomPorts
gradle buildProduct
```
